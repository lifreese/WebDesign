Sites to Find Old Games!

  • Itch.io is worth a look to see if someone has made an old game playable! For example, here's an updated version of Bugdom!
  • My Abandonware is a great resource to find old games to play. However, it should be noted that these will take a little more installation set up on your end as opposed to itch.io's runnable files. For example, here's a playable version of Zoo Tycoon!
  • The Video Game History Foundation is a digital library of gaming media, think magazines etc. While I haven't been able to find any playable games, you can still read up on them or look at their source code!
